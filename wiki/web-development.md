# Web Development

Web development encompasses all the tasks involved in building websites and web applications for the internet or an intranet. It ranges from creating simple static pages to complex web applications, electronic businesses, and social network services.

## The Web Stack

Modern web development is built on several foundational technologies:

- **HTML** (HyperText Markup Language): Defines the structure and content of web pages
- **CSS** (Cascading Style Sheets): Controls the visual presentation and layout
- **JavaScript**: Adds interactivity and dynamic behavior

## Frontend vs Backend

**Frontend** (client-side) development focuses on what users see and interact with in their browser. It involves HTML, CSS, JavaScript, and frameworks like React, Vue, or Angular.

**Backend** (server-side) development handles data processing, business logic, authentication, and database operations. Common technologies include Node.js, Python (Django/Flask), Ruby on Rails, and Go.

## Modern Web Architecture

### REST APIs

Representational State Transfer (REST) is an architectural style for designing networked applications. RESTful APIs use HTTP methods (GET, POST, PUT, DELETE) to perform CRUD operations on resources.

### Single Page Applications (SPAs)

SPAs load a single HTML page and dynamically update content as the user interacts with the app. Frameworks like React, Vue, and Angular enable this approach, providing faster user experiences.

### Databases

- **Relational (SQL)**: PostgreSQL, MySQL, SQLite for structured data with relationships
- **NoSQL**: MongoDB, Redis, DynamoDB for flexible schemas and high scalability

## Deployment

Modern deployment practices include containerization with Docker, CI/CD pipelines with GitHub Actions or Jenkins, cloud hosting on AWS/GCP/Azure, and static site hosting on platforms like Vercel, Netlify, and GitHub Pages.
